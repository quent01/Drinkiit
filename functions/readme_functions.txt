Il contiendra toutes les fonctions dont nous aurons besoin pour que le site fonctionne.
Chaque fichier de fonctions contiendra un ensemble de fonctions propre.
Par exemple, un fichier de fonction "article.fn.php" contiendra toutes les fonctions liées aux articles (ajout, suppression, sélection).

Il contiendra les fichiers:
- article.fn.php
- database.fn.php
- user.fn.php
